🚀 MinesTower<br />
MinesTower é um jogo interativo desenvolvido com uma interface gráfica atraente e uma lógica de programação envolvente. O jogo desafia os usuários a aumentar seu saldo, com a opção de ganhar ou dar cashout a qualquer momento.<br />

🎯 Funcionalidades<br />
Saldo Inicial: O usuário começa com um saldo para iniciar suas jogadas.<br />
Lógica de Jogo: A cada rodada, o usuário pode aumentar seu saldo conforme avança, ou optar por dar cashout para garantir seus ganhos.<br />
Cashout: Função que permite ao jogador encerrar a partida e retirar os ganhos a qualquer momento.<br />
Interface Gráfica Atraente: Um design visual bonito que proporciona uma experiência de usuário imersiva.<br />

💻 Tecnologias Utilizadas<br />
React: Biblioteca JavaScript para construção de interfaces de usuário.<br />
Lógica de Programação: Algoritmos que garantem o comportamento dinâmico e envolvente do jogo.<br />
Bootstrap 5: Framework CSS para garantir uma interface responsiva e moderna.<br />
